# Generation Review SOP

## Purpose

Review generated music outputs using stable criteria instead of taste drift.

## Review Dimensions

1. Prompt adherence
2. Hook clarity
3. Structure coherence
4. Vocal or lead quality
5. Emotional fit
6. Deliverability

## Output Labels

1. `keep`
2. `revise`
3. `reject`

## Writeback Rule

If a repeated review rule emerges, add it to the project knowledge base or update this SOP.
