# Example: AI Music Project

This example shows how Project-Agent-OS can be applied to a generic AI music workflow without tying it to any specific brand, product, or campaign.

这个示例展示了 `Project-Agent-OS` 如何应用在一个通用 AI 音乐工作流中，而不绑定任何特定品牌、产品或营销项目。

## Example Scope

The project agent supports a small AI music operation that may include:

1. Idea intake
2. Prompt design
3. Model routing
4. Generation review
5. Packaging
6. Delivery formatting
7. Archive and learnings
8. Asset database upkeep

这个示例的重点不是定义主仓库身份，而是展示：

1. AI 音乐工作流如何使用本地 memory
2. 生成与归档规则如何沉淀成 SOP
3. 示例层如何承载更具体、更领域化的操作规则

## Example Structure

```text
ai-music-project/
  knowledge-base/
  sop/
  thread-summaries/
  scripts/
```

## Example Principle

The workspace owns the process.

The threads do the work.

The memory keeps the project from forgetting.

For repeated T8Star/Suno work, archive discipline matters as much as prompting quality.

本示例中的 T8Star/Suno 规则属于示例层经验，不自动等于主仓库的通用方法论。
